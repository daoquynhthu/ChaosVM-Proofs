# 端到端安全定理：Φₜ 共轭执行定理

## 形式化陈述

```
∀ (program : EncodedProgram) (input : InitialState) 
  (R₁ R₂ : R_run) (P₁ P₂ : Poison):

  let r₁ := run(program, input, R₁, P₁)
  let r₂ := run(program, input, R₂, P₂)

  // ── Claim 1: 功能等价性（语义不变）──
  // 当投毒为零时，任意 R_run 产生相同最终状态
  (P₁ = 0 ∧ P₂ = 0) → r₁.final_state = r₂.final_state

  // ── Claim 2: 反篡改发散（投毒 → 错误输出）──
  // 投毒程序最终输出 ≠ 干净程序输出
  (P₁ = 0 ∧ P₂ ≠ 0) → r₁.final_state ≠ r₂.final_state

  // ── Claim 3: 运行间多样性（不同轨迹）──
  // 不同 R_run 产生不同的执行痕迹
  (R₁ ≠ R₂ ∧ P₁ = P₂) → r₁.trace ≠ r₂.trace
```
